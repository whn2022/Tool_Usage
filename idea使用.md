## 一、快捷键

Command + shift + f 全局搜索

Command + shift+ r 全局替换

Command + Delete 删除光标所在行

Command + x 剪切当前行

Command + y 复制当前行

Command + f 在当前文件搜索

Command + option + 左箭头 返回上一步光标的所在位置

Command + 7 查看当前类的结构



Option + 向上箭头/向下箭头 选中很多需要的行/不需要的行
